# Online Pharmacy Information System

## Project Overview
Information system for managing a chain of online pharmacies with role-based access and automated business processes.

## Features

### Customer
- Browse and search medicines
- Place orders as guest users
- Alternative pharmacy suggestions

### Employee
- Process orders and update status
- Manage inventory and stock levels
- View assigned pharmacy operations

### Manager
- Manage employees and pharmacies
- Control suppliers and medicine catalog
- Create purchase orders and reports
- Schedule employee shifts

## Business Logic
- Automatic stock updates on orders
- Stock replenishment after delivery
- Role-based access control
- Automated order lifecycle

## Output
- Sales reports
- Inventory reports
- Order tracking system
